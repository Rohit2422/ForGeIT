Broken Link Checker

  • The tool looks through the current webpage and collects all the clickable links (<a> elements). Links that are part of footnotes are ignored to keep results clean.
  • A small floating panel appears on the screen. It is draggable, so you can move it around. The panel has three sections: all links being checked, links that work, and links that are broken.
  • The tool tests every collected link by contacting the server. If the server responds correctly, the link is marked as working (green). If the server cannot be reached, or the link is invalid, it is marked as broken (red). Local or incomplete links are also treated as broken.
  • As each link is checked, the results show up in real time inside the panel. Links are color-coded, and you can click any link in the list to jump to its place on the page, where it will briefly highlight.
  • The panel includes a refresh button to re-run the check and a close button to remove it. Visual feedback like a spinner shows when checks are in progress, and broken links flash for extra attention.
